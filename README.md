# LittleFS C2000
This is an example LittleFS project.  TMS320F28335 is used as a target microcontroller.
This microcontroller has a byte definition of 16 bits.

```
sizeof(byte)  == 1
sizeof(short) == 1
sizeof(long)  == 2
```
